# Blurry and Similar Images Remover (BnSIR)

**Supprimez automatiquement les images floues et les doublons similaires**

BnSIR est un outil conçu pour nettoyer vos collections d'images en supprimant :
- Les images **trop floues**.
- Les **doublons** ou images trop similaires (en ne gardant qu'une seule copie).

---

## Fonctionnalités
- Détection et suppression des images floues.
- Identification et suppression des images similaires ou identiques.
- Code source libre d'utilisation pour vos projets.
