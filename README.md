# IP Scanner Tool

**IP Scanner Tool** est une application légère développée en <keyword>Python</keyword> dotée d'une interface graphique (<keyword>GUI</keyword>) permettant de scanner rapidement un réseau local. Elle identifie les machines actives, récupère leurs noms d'hôte (<keyword>hostname</keyword>) et leurs adresses <keyword>MAC</keyword>.

## 1. Fonctionnalités principales

- **Détection Automatique du Réseau** : Identifie l'adresse IP locale et le masque de sous-réseau pour définir la plage de scan.
- **Scan Multithread** : Utilise <keyword>ThreadPoolExecutor</keyword> pour effectuer des requêtes <keyword>ICMP</keyword> (ping) en parallèle, optimisant ainsi la vitesse d'exécution.
- **Résolution d'Identité** :
    - Récupération du nom d'hôte via <keyword>socket</keyword>.
    - Récupération de l'adresse MAC via des requêtes <keyword>ARP</keyword> (Address Resolution Protocol).
- **Interface Intuitive** : Affichage des résultats dans un tableau structuré (`Treeview`).
- **Interaction** : Possibilité de copier les données (IP, Hostname ou MAC) directement dans le presse-papier via un menu contextuel (clic droit).

## 2. Architecture Technique

L'application repose sur plusieurs bibliothèques spécialisées :
- **<keyword>netifaces</keyword>** : Pour l'extraction des configurations réseau de l'hôte.
- **<keyword>pythonping</keyword>** : Pour la gestion des paquets ICMP.
- **<keyword>Scapy</keyword>** : Pour la forge et l'envoi de trames <keyword>Ethernet</keyword>/ARP de bas niveau.
- **<keyword>Tkinter</keyword>** : Pour la construction de l'interface utilisateur.
- **<keyword>concurrent.futures</keyword>** : Pour la gestion de la programmation asynchrone (multithreading).

## 3. Prérequis

Le script nécessite des privilèges d'administrateur (ou `sudo` sous Linux/macOS) pour l'envoi de paquets réseau bruts (<keyword>Raw Sockets</keyword>) via Scapy et Pythonping.

### Dépendances
```bash
pip install netifaces pythonping scapy
```

## 4. Utilisation

1. Lancez l'application :
   ```bash
   python main.py
   ```
2. Cliquez sur le menu **Run** pour démarrer le scan du réseau local.
3. Les résultats s'afficheront dynamiquement dans la liste.
4. Faites un **clic droit** sur une cellule pour copier sa valeur.

## 5. Logique de Scan de Sous-réseau

Le script calcule la plage d'adresses à scanner en fonction du masque de sous-réseau détecté :
- **255.255.255.0 (Classe C)** : Scan de `x.x.x.1` à `x.x.x.254`.
- **255.255.0.0 (Classe B)** : Scan étendu sur les deux derniers octets.
- **255.0.0.0 (Classe A)** : Scan global sur les trois derniers octets.
